# s21_decimal — Реализация типа данных Decimal

## 📚 О проекте

Этот проект представляет собой реализацию собственной библиотеки для работы с десятичными числами фиксированной точности — тип `decimal`, отсутствующий в стандарте языка C.

Тип `decimal` критически важен для финансовых расчетов, где недопустимы погрешности вычислений, свойственные типам с плавающей точкой (IEEE 754). Библиотека обеспечивает точные арифметические операции, сравнения, преобразования и математические округления.

В ходе проекта я реализовал:
- Полный набор арифметических операций (сложение, вычитание, умножение, деление)
- Операторы сравнения
- Преобразователи (int/decimal/float)
- Функции округления и работы с десятичными дробями

Библиотека разработана на языке C стандарта C11, оформлена как статическая и покрыта unit-тестами.

typedef struct {

bits[0], bits[1], bits[2] — младшие, средние и старшие 32 бита 96-разрядного целого числа

bits[3] — знак (бит 31) и показатель степени (биты 16-23)
