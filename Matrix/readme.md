# s21_matrix — Реализация библиотеки для работы с матрицами

## 📚 О проекте

Этот проект представляет собой реализацию собственной библиотеки для обработки числовых матриц на языке C.

Матрицы являются одной из базовых структур данных в программировании. Они применяются для представления табличных значений, для вычислительных задач, компьютерной графики и нейронных сетей. В рамках этого проекта я реализовал полный набор операций для работы с матрицами — от базовых арифметических действий до вычисления определителя и обратной матрицы.

Библиотека разработана на языке C стандарта C11, оформлена как статическая и покрыта unit-тестами.
