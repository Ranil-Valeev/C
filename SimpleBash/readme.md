# Simple Bash Utils — Разработка утилит cat и grep

## 📚 О проекте

Этот проект представляет собой реализацию двух базовых утилит Bash для работы с текстом: **cat** и **grep**.

Утилиты разработаны на языке C стандарта C11 с использованием компилятора gcc. В ходе проекта я изучил внутреннее устройство этих инструментов, освоил работу с файловой системой на низком уровне, научился обрабатывать аргументы командной строки и работать с регулярными выражениями.
