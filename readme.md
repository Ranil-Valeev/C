# 🐧 Программирование на C

## 📚 О репозитории

Этот репозиторий содержит проекты, выполненные в рамках обучения программированию на языке C.

Здесь представлены:
- Задачи с **интенсива** (отработка базовых навыков)
- Реализации классических утилит Unix (`cat`, `grep`)
- Собственные библиотеки для работы со строками (`string.h` с расширениями)
- Тип данных для точных финансовых вычислений (`decimal`)
- Библиотека для работы с матрицами

Все проекты разработаны на языке C стандарта C11, оформлены как статические библиотеки (где применимо) и покрыты тестами.

---

## 📋 Проекты

| № | Папка | Название | Описание |
|---|-------|----------|----------|
| 0 | `intensiv` | Задачи интенсива | Базовые алгоритмические задачи на C |
| 1 | `simplebash` | Simple Bash Utils | Реализация команд `cat` и `grep` |
| 2 | `stringplus` | s21_string+ | Реализация `string.h` + `sprintf` + `sscanf` |
| 3 | `decimal` | s21_decimal | Тип данных для точных финансовых расчетов |
| 4 | `matrix` | s21_matrix | Библиотека для работы с матрицами |
