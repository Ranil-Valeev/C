# s21_string+ — Реализация библиотеки string.h с дополнениями

## 📚 О проекте

Этот проект представляет собой реализацию собственной библиотеки для обработки строк на языке C, аналогичной стандартной `string.h`, но с расширенным функционалом.

В ходе проекта я реализовал:
- Все основные функции стандартной библиотеки `string.h`
- Собственную версию функций `sprintf` и `sscanf` с поддержкой форматирования
- Дополнительные функции обработки строк (преобразование регистра, вставка, обрезка)

Библиотека разработана на языке C стандарта C11, оформлена как статическая и покрыта unit-тестами с проверкой через библиотеку Check.

Флаги: -, +, (пробел)

Ширина: (число)

Точность: .(число)

Длина: h, l
